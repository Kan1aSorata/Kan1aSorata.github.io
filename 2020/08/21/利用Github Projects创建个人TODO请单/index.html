利用GitHub Projects创建个人TODO清单

写在前面：第二次编写教程，如发现有任何的错误，欢迎点击关于的联系方式联系我~

登入GitHub并创建你的Project

很简单，进入GitHub个人页面之后点击首页导航栏中的 + -> New project

进入后更改Project board name, Project template和Visibility，分别用于你的项目名称、选择项目模版和隐私权限设置（即谁能看到这个project，Project template选择Automated kanban

最后点击Create project。

创建TODO

进入project后的页面是这样的由于你选择了预设的模版已经给你创建好了三列空间

你可以在这些空间内新建多个TODO块进行分类，方法很简单，直接点击Column上的 + 即可

若需要更多的Column，请点击最右侧的Add a Column

若要对当前Column/Card名称进行编辑，点击对应的 ···

编辑Note

GitHub projects支持markdown，我会把我常用的markdown补充在下面便于参考
